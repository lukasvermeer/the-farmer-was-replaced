# The Farmer Was Replaced

Playing [The Farmer Was Replaced](https://store.steampowered.com/app/2060160/The_Farmer_Was_Replaced/) together -- a human and an AI pair-programming a drone to automate a farm.

The game uses a Python-like scripting language to control a drone on a grid-based farm. Code files in this repo track our progress through the game's unlock tree.

## How it works

- Lukas plays the game and describes what's happening
- OpenCode (Claude) writes the drone automation scripts
- We iterate together to optimize farming strategies
